# 화면 캡처 가이드

GitHub README와 방송 사전 자료에 사용할 권장 화면 목록입니다.

## 캡처 기준

- 데스크톱: 1440×900
- 모바일: 390×844
- 브라우저 확대: 100%
- 파일 형식: PNG
- 개인정보가 들어간 화면은 샘플 이름 `홍길동` 사용
- 사진은 사용 동의를 받은 샘플만 사용
- 캡처 전 새로고침 후 애니메이션이 멈춘 시점을 선택

## README 배치

1. 최상단에는 네 제품의 결과 화면을 한 장씩 사용합니다.
2. 각 제품 설명에는 인트로, 핵심 인터랙션, 결과 순서로 세 장을 배치합니다.
3. 모바일 화면은 상세 문서에 두고 README 본문은 데스크톱 중심으로 구성합니다.
4. GIF는 핵심 전환 한 개만 사용하며 10MB 이하로 유지합니다.

## 현재 포함된 이미지

`docs/images/`에는 네 제품의 최신 대표 비주얼을 선별해 두었습니다. SUPEX를 현실로 원본 장면과 생성 매니페스트는 `sk-supex-3/public/assets/supex-book/`에 있습니다. UI 캡처 파일은 위 이름으로 추가하면 README 이미지와 쉽게 교체할 수 있습니다.
