# 방송·현장 시연 가이드

세 제품을 같은 방송에서 보여줄 때의 권장 흐름입니다.

## 준비

- 1440×900 이상 데스크톱 브라우저
- 안정적인 네트워크
- SUPEX LICENSE에 사용할 정면 얼굴 사진 1장
- 브라우저 확대 100%
- 알림과 자동 번역 기능 끄기
- 촬영 전 세 운영 URL을 각각 새 탭에 열기

| 탭 | URL |
| --- | --- |
| 나의 SUPEX | <https://sk-supex.vercel.app> |
| SUPEX LICENSE | <https://sk-supex-1.vercel.app> |
| SUPEX RUN | <https://sk-supex-2.vercel.app> |

## 권장 시연 구성

### 1 · 발견: 나의 SUPEX

권장 시간: 60~90초

1. `나의 SUPEX 찾기`를 선택합니다.
2. 첫 질문에서 선택지에 마우스를 올려 장면이 바뀌는 모습을 보여줍니다.
3. 네 질문에 응답합니다.
4. 마지막 선택 뒤 네 장면과 `SUPEX` 글자가 완성되는 과정을 기다립니다.
5. 우하단 화살표를 눌러 결과로 이동합니다.
6. 타입 이름과 다음 행동을 읽습니다.
7. `다른 성향 보기`로 12개 타입을 한 화면에서 보여줍니다.

전달할 핵심:

> 같은 상황에서도 무엇을 먼저 선택하는지에 따라 SUPEX를 만드는 방식이 달라집니다.

### 2 · 진단: SUPEX LICENSE

권장 시간: 80~120초

1. 이름, 회사, 직무를 입력합니다.
2. 네 문항에 가장 가까운 현재 행동을 선택합니다.
3. 네 개의 원형 점수를 보여줍니다.
4. 가장 강한 SUPEX와 행동 해석을 확인합니다.
5. 얼굴 사진을 업로드합니다.
6. 라이선스 발급을 누르고 네 가지 빛이 각인되는 장면을 보여줍니다.
7. 홀로그램 카드를 마우스로 움직입니다.
8. 색상을 바꾸고 PNG 저장 기능을 보여줍니다.

전달할 핵심:

> 네 점수는 현재 업무에서 기준, 협업, 적용, 실행을 얼마나 구체적으로 정하는지 보여줍니다.

촬영 전 확인:

- AI 포트레이트 생성은 네트워크 상태에 영향을 받습니다.
- 발급용 사진은 얼굴이 선명하고 정면에 가까운 이미지를 사용합니다.
- 생성 실패에 대비해 완성된 라이선스 PNG를 백업 화면으로 준비합니다.

### 3 · 체험: SUPEX RUN

권장 시간: 60~75초

1. `시작하기`를 선택합니다.
2. 탁월 게임에서 바깥 원부터 기준선에 맞춥니다.
3. 단합 게임에서 손을 떼지 않고 1부터 7까지 연결합니다.
4. 적용 게임에서 가운데 단어와 같은 단어를 고릅니다.
5. 실행 게임에서 1부터 10까지 순서대로 누릅니다.
6. 종합 점수와 코멘트를 보여줍니다.

전달할 핵심:

> SUPEX의 네 가지 힘은 높은 기준, 연결된 흐름, 빠른 반영, 끝까지 실행하는 행동으로 나타납니다.

## 4분 통합 진행안

| 시간 | 화면 | 진행 |
| --- | --- | --- |
| 0:00–0:20 | 세 제품 소개 | 발견, 진단, 체험의 구조 설명 |
| 0:20–1:30 | 나의 SUPEX | 선택 장면과 타입 결과 시연 |
| 1:30–2:50 | SUPEX LICENSE | 네 점수와 홀로그램 발급 시연 |
| 2:50–3:50 | SUPEX RUN | 대표 스테이지와 최종 점수 시연 |
| 3:50–4:00 | 마무리 | 나의 방식에서 실제 행동까지 연결 |

## 조작 요약

| 제품 | 주요 조작 | 보조 조작 |
| --- | --- | --- |
| 나의 SUPEX | 마우스·터치 선택 | 숫자 1~3 선택, `R` 재시작, `B` 전체 화면 |
| SUPEX LICENSE | 입력·선택·사진 업로드 | 카드 틸트, 색상 변경, PNG 저장 |
| SUPEX RUN | 마우스·터치 게임 | `P` 일시정지 |

## 현장 복구

- 화면 비율이 어색하면 브라우저 확대를 100%로 맞춥니다.
- 제품이 멈추면 새로고침 후 처음부터 시작합니다.
- SUPEX RUN은 `다시 시작`으로 새 게임을 바로 만들 수 있습니다.
- 나의 SUPEX는 `R` 키로 인트로로 돌아갈 수 있습니다.
- SUPEX LICENSE의 AI 생성이 실패하면 점수 화면에서 사진을 다시 업로드합니다.
